## Workshop - 一天實作 Ubike Map 就上手！程式實作工作坊

### 課程內容
- 製作縣市/地區下拉選單
- 使用 AJAX 取得遠端資料
- 使用 OSM 和 Leaflet 製作地圖與標記 Ubike 站點
    - 建立地圖
    - 疊加 Ubike 站點標記
    - 移除舊標記
    - 顯示 Ubike 數量資訊
    - 畫面隨下拉選單結果移至該區域
- 將實作發佈到 GitHub page
